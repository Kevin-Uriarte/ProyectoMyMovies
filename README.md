## Descripción

Este proyecto consiste en el desarrollo de una aplicación web utilizando Django, desplegada en AWS, que permite la gestión y visualización de películas.

Se implementaron funcionalidades como:

* Registro y visualización de películas
* Sistema de reseñas y calificaciones
* Interacción mediante likes y comentarios
* Panel de administración para gestión de datos
* Conexión a base de datos PostgreSQL

El sistema fue configurado con un entorno virtual (venv) y desplegado en un servidor EC2, permitiendo su acceso a través de internet.
